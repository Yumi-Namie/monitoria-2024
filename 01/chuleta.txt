cd - cambiar de directorio

rm - borrar

code - abrir el fichero en vscode
touch - crear archivo nuevo

git branch clase01: chuleta, desafio, funciones, password

main - principal - solo archivos buenos (todo el equipo)

git merge nombre_rama

//Borrar rama:
